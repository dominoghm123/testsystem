# PRD：中职电子电工题库（MVP）

- 版本：v1.0
- 日期：2026-04-04
- 目标上线：2026-04-08（周三）
- 产品形态：Windows 离线桌面应用（Electron）

## 1. 背景与目标

面向中等职业高中高三电子电工专业学生，服务职业高中电子电工高考复习。MVP 目标是在无外网环境下提供稳定、可练习、可模拟考试、可讲解解析的题库应用。

## 2. 目标用户

1. 学生：进行专项练习、模拟考试、错题回看。
2. 教师：录入题目、绑定题图、发布题库、维护题库质量。

## 3. 教学范围

- 参考教材：
  - 《电子技术基础与技能（第3版）》张金华
  - 《高职考电子电工类专业总复习》郑秀萍
- 首批题量：40 题
- 页面展示：每页 4 题

## 4. 核心 KPI

1. 下周三前可在目标电脑稳定运行（离线）。
2. 首批 40 题可完整练习与模拟考。
3. 题库发布过程零静默失败（发布失败可回滚，且有日志）。
4. 教师/学生模式支持一键切换（教师端有 PIN 保护）。

## 5. In Scope（MVP 必做）

### 5.1 学生端

1. 练习模式：每页 4 题，支持单选/判断（以题库配置为准）。
2. 模拟考模式：限时、交卷、自动评分。
3. 错题回看：按最近一次考试/练习记录筛选。
4. 题图展示：静态 SVG（题干与题图不交互）。
5. 解析区：动画交互式说明（亮灯、电流流向、开关状态等）。

### 5.2 教师端

1. PIN 进入教师模式。
2. 可视化录题（题干、选项、答案、解析参数、题图路径）。
3. 批量导入/导出题库。
4. 发布题库前强校验（字段、资源、模板参数）。
5. 版本化原子发布与自动回滚。

### 5.3 部署与迁移

1. Windows 单平台打包（`.exe`）。
2. 应用与数据目录可复制到另一台无外网电脑运行。
3. 默认本地数据：题库 JSON + 作答 SQLite。

## 6. Out of Scope（本期不做）

1. 账号系统与云端同步。
2. 教师后台联网服务。
3. macOS/Linux 打包。
4. 题图题干实时交互编辑器（仅解析动画交互）。

## 7. 非功能要求

1. 稳定性：关键流程失败可见（错误码 + 本地日志）。
2. 安全性：教师端 PIN、导入文件白名单与大小限制。
3. 可维护性：解析动画模板驱动，不为每题硬编码逻辑。
4. 轻量上线：最小依赖、最小模块数量、可快速交付。

## 8. 数据与目录约定

1. 题库文件：`question-bank/bank_vNN.json`
2. 当前版本指针：`question-bank/current.json`
3. 作答数据库：`data/app.db`
4. 发布/导入日志：`logs/publish.log`、`logs/import.log`

## 9. 验收标准（Definition of Acceptance）

1. 学生端可完整完成 40 题练习与模拟考流程。
2. 每页固定 4 题，分页切换不丢答题状态。
3. 至少 1 套解析动画模板可覆盖首批题目。
4. 教师端发布时，错误题库无法进入学生端。
5. 发布失败可自动回滚到上一稳定题库版本。
6. 拷贝到另一台 Windows 电脑可离线启动并使用。

## 10. 风险与缓解

1. 风险：题库格式不一致导致上线前阻塞。
   - 缓解：提前定义 Schema + 导入校验报告。
2. 风险：解析动画模板覆盖不足。
   - 缓解：首批只支持有限标准模板并约束题型。
3. 风险：打包后路径差异导致资源找不到。
   - 缓解：统一相对路径 + 启动完整性检查。
4. 风险：SQLite 锁冲突。
   - 缓解：写入重试、串行提交、失败可见提示。
